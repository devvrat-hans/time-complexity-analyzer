Time Complexity Guide

Complexity Name Description Examples
O(1) Constant Same time regardless of input size Array access, hash lookup
O(log n) Logarithmic Time grows logarithmically Binary search, tree operations
O(n) Linear Time grows linearly with input Linear search, single loop
O(n log n) Linearithmic Efficient sorting complexity Merge sort, heap sort
O(n²) Quadratic Time grows with square of input Bubble sort, nested loops
O(2^n) Exponential Time doubles with each input Recursive fibonacci, subsets

Code Examples

Example 1: Linear Search - O(n)
int linearSearch(vector<int>& arr, int target) {
    for (int i = 0; i < arr.size(); i++) {
        if (arr[i] == target) {
            return i;
        }
    }
    return -1;
}
Example 2: Binary Search - O(log n)
int binarySearch(vector<int>& arr, int target) {
    int left = 0, right = arr.size() - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
Example 3: Bubble Sort - O(n²)
void bubbleSort(vector<int>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
            }
        }
    }
}
